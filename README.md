# Weather_App_with_Open_Weather_API_and_Docker

This project consists of creation a Python-based application that utilizes the Open Weather API to retrieve real-time weather data, leveraging Docker to run the web application and store the data in a NoSQL database (MongoDB). Additionally, users will be able to create accounts and subscribe to specific countries to receive notifications and alerts about the weather. To achieve this functionality, we utilized Kafka that will send an email notification to the appropriate user.
